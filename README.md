## textlintによる文章チェック
`/` にて下記のコマンドを入力する事により文章チェックを行うことができる。<br>
- この機能はローカルマシンに `Node.js` をインストールする必要がある。<br>
[こちらの記事](https://qiita.com/ryome/items/eec08b28aff294e8c3d6) などご参考にインストールをお願いします。

```bash
npm run lint
```
